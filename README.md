# To-Do List Web App

A simple and interactive To-Do List built using *HTML, **CSS, and **JavaScript*.

## Features

- Add new tasks
- Mark tasks as completed
- Remove tasks from the list

## Live Demo

Open [Vercel](https://https://elevate-labs-task2.vercel.app/) for live demo

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- DOM Manipulation
- Event Handling
